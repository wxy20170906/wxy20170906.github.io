    Git依旧

      今天是进入兄弟会的第6天，今天我学了许多git有关的命令，学会了怎么样去合并分支
    下面是步骤

    1. git clone git@github.com:17824909342/hellocountry.git
    2. cd hellocountry
    3. git branch 分支名
    4. 修改下库文件
    5. git add .
    6. git commit -m”名字”
    7. git push origin sb
    8. git checkout master
    9. git merge sb
